# Part 3 - File I/O, APIs & Exception Handling

This project focuses on external data interaction and robust error handling.

### Features:
1. **Task 1 (File I/O):** Creates and manages `python_notes.txt` with write and append modes.
2. **Task 2 (API Integration):** Fetches real-time data from DummyJSON, featuring filtered and sorted results.
3. **Task 3 (Exception Handling):** Implements a guarded calculator and a robust input validation loop.
4. **Task 4 (Logging):** A persistent `error_log.txt` that records timestamped errors and API failures.
