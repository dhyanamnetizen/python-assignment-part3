# Part 3 - File I/O, APIs & Exception Handling

This project focuses on external data interaction and robust error handling.

### Features:
1. **File I/O:** Createing and managing `python_notes.txt` with write and append modes.
2. **API Integration:** Fetching real-time data from DummyJSON, featuring filtered and sorted results.
3. **Exception Handling:** Implementing a guarded calculator and a robust input validation loop.
4. **Logging:** A persistent `error_log.txt` recording timestamped errors and API failures.
